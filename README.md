# 一箪食 yidanshi

> 一箪食，一瓢饮，在陋巷……不改其乐。——《论语·雍也》

记录自己做的每一顿饭，沉淀成一份好看的个人食单。

你是不是也这样：每周做饭，做法都是现找教程，做完就忘了怎么做的？

一箪食帮你把「做过的饭」变成资产：

- 📷 **拍照即记录** — 做完饭拍一张，自动抠出碗盘、合成深色底菜卡（本地 [rembg](https://github.com/danielgatis/rembg) + BiRefNet，不花钱不上传云端）
- 📖 **自己的菜单** — 分类浏览（一碗饭 / 一碗面 / 一碗汤 / 一碗菜），每道菜有品味评分、做过几次、做法教程
- 🎨 **插画风教程卡** — 食材图标 + 分步卡通插图的做法卡片，照着自己的教程做，不再翻遍收藏夹
- 🤖 **AI 友好的数据** — 菜谱和记录存为本地 Markdown/JSON，把社交媒体上看到的教程丢给 AI 助手（Claude / Codex 等）即可整理录入，也可页面上手动编辑
- 🏠 **自托管** — 跑在自己电脑上，手机浏览器加主屏当 App 用

灵感来自微信小程序「我的Taste」（by 抖音博主 AB）——它展示的是作者自己的菜谱，而一箪食让每个人都能拥有自己的版本。
