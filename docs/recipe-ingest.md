# 把社交媒体教程录入一箪食

数据就是文件：一道菜 = `data/recipes/<id>.md`。给 AI 助手（Claude / Codex 等）
下面这段指令 + 教程原文（链接、文字、截图皆可），它写出的文件放进目录即完成录入。

## 给 AI 的指令模板

> 把下面这个做菜教程整理成一箪食的菜谱文件，写入 `data/recipes/<拼音或英文id>.md`，格式：
>
> ```markdown
> ---
> id: lusun-niupai
> name: 芦笋牛排
> category: 一碗菜        # 一碗饭 / 一碗面 / 一碗汤 / 一碗菜 / 一碗甜
> source: "https://原教程链接"
> created: 2026-07-17
> ---
>
> ## 食材
>
> - 芦笋 | 一把
> - 牛排 | 1块
> - 蒜 | 4瓣
>
> ## 步骤
>
> 1. 牛排切条吸干水分，加黑胡椒、生抽、蚝油腌制10分钟。
> 2. 芦笋切段焯水40秒备用。
> 3. 热锅下蒜和牛排煎至金黄，加入芦笋炒到喜欢的熟度，黑胡椒海盐调味出锅。
>
> ## 贴士
>
> - 可以按照个人理解和口味调整。
> ```
>
> 要求：步骤合并成 3–6 步、每步一句话说清楚；食材「名称 | 用量」；保留原教程链接到 source。

## 配套插画（可选）

```bash
.venv/bin/python scripts/gen_illust_prompts.py <recipe-id>
```

会生成 `data/photos/illust/<recipe-id>/prompts.md`（统一画风的食材/步骤插画 prompt 清单），
交给图像生成工具出图后按文件名（`ing-1.png`、`step-1.png`…）放回同目录，教程卡自动显示插画；
没有插画时教程卡以 emoji 图标 + 纯文字步骤降级显示，同样可用。
